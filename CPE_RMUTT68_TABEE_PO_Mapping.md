# TABEE PO–PLO Mapping

> ดูข้อมูลหลักสูตรและ PLO definitions: [CPE_RMUTT68_Prospectus.md](CPE_RMUTT68_Prospectus.md) | ดูการแมป PLO–CLO ทุกรายวิชา: [CPE_RMUTT68_PLO_CLO_Matrix.md](CPE_RMUTT68_PLO_CLO_Matrix.md)

## หลักสูตรวิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมคอมพิวเตอร์ มรท. (ปรับปรุง พ.ศ. 2568)

ตารางนี้แสดงความเชื่อมโยงระหว่าง TABEE Program Outcomes (TABEE PO) ทั้ง 11 ข้อ ตาม Sydney Accord กับ PLOs ของหลักสูตร สำหรับ:

- **อาจารย์ผู้สอน** — ใช้อ้างอิงเมื่อออกแบบ CLOs ของรายวิชาให้ครอบคลุม TABEE PO ที่เกี่ยวข้อง
- **ผู้ประเมิน TABEE** — ใช้ตรวจสอบว่า PLOs ของหลักสูตรครอบคลุม TABEE PO ครบทั้ง 11 ข้อ

อ้างอิงคำอธิบาย TABEE PO จาก `source_documents/LO Writing Manual TABEE PO-PI-CLO ENG RMUTT.pdf`

---

## ตารางแมป TABEE PO ↔ PLO

| TABEE PO | หัวข้อ | PLO ที่เกี่ยวข้อง | ผลลัพธ์การเรียนรู้ระดับหลักสูตร (PLO) |
| --- | --- | --- | --- |
| **PO1** | ความรู้ทางวิศวกรรม / พื้นฐานคณิตศาสตร์และวิทยาศาสตร์ (Engineering knowledge) | [PLO1](CPE_RMUTT68_Prospectus.md#plo1) | อธิบายและคำนวณหลักการสำคัญทางคณิตศาสตร์พื้นฐาน วิทยาศาสตร์พื้นฐาน วิศวกรรมศาสตร์พื้นฐาน และวิศวกรรมคอมพิวเตอร์พื้นฐานได้ |
| **PO2** | การวิเคราะห์ปัญหาทางวิศวกรรม (Problem analysis) | [PLO4](CPE_RMUTT68_Prospectus.md#plo4) | คิดวิเคราะห์อย่างเป็นระบบ สืบค้นและทดลองเพื่อพัฒนาระบบงานทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างเป็นขั้นตอน |
| **PO3** | การออกแบบและพัฒนาเพื่อหาคำตอบของปัญหา (Design/development of solutions) | [PLO2](CPE_RMUTT68_Prospectus.md#plo2), [PLO3](CPE_RMUTT68_Prospectus.md#plo3) | PLO2: ใช้ความรู้และทักษะ ในเชิงทฤษฎีและปฏิบัติ เพื่อแก้ไขปัญหาด้านวิศวกรรมคอมพิวเตอร์ได้ · PLO3: บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาให้กับอุตสาหกรรมหรือชุมชน เพื่อการพัฒนาที่ยั่งยืน |
| **PO4** | การพิจารณาตรวจสอบ (Investigation) | [PLO4](CPE_RMUTT68_Prospectus.md#plo4), [PLO7](CPE_RMUTT68_Prospectus.md#plo7) | PLO4: คิดวิเคราะห์อย่างเป็นระบบ สืบค้นและทดลองเพื่อพัฒนาระบบงานทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างเป็นขั้นตอน · PLO7: อธิบายและประยุกต์ใช้เทคโนโลยีทางวิศวกรรมคอมพิวเตอร์ที่ไม่เคยเรียนในหลักสูตรได้ |
| **PO5** | การใช้เครื่องมือทันสมัย (Modern tool usage) | [PLO5](CPE_RMUTT68_Prospectus.md#plo5) | ใช้เครื่องมือทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างถูกต้องเหมาะสม |
| **PO6** | การทำงานร่วมกันเป็นทีม (Team work) | [PLO8](CPE_RMUTT68_Prospectus.md#plo8) | ทำงานเป็นทีม วางแผนและบริหารจัดการงานที่ได้รับมอบหมาย สื่อสารความคืบหน้าและปัญหาได้ตรงประเด็น และเสนอแนวทางแก้ไขพร้อมเหตุผลรองรับได้ |
| **PO7** | การติดต่อสื่อสาร (Communication) | [PLO8](CPE_RMUTT68_Prospectus.md#plo8) | ทำงานเป็นทีม วางแผนและบริหารจัดการงานที่ได้รับมอบหมาย สื่อสารความคืบหน้าและปัญหาได้ตรงประเด็น และเสนอแนวทางแก้ไขพร้อมเหตุผลรองรับได้ |
| **PO8** | ความรับผิดชอบของวิศวกรต่อโลก (Ethics/sustainability) | [PLO3](CPE_RMUTT68_Prospectus.md#plo3) | บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาให้กับอุตสาหกรรมหรือชุมชน เพื่อการพัฒนาที่ยั่งยืน |
| **PO9** | จรรยาบรรณแห่งวิชาชีพ (Professional ethics) | [PLO6](CPE_RMUTT68_Prospectus.md#plo6) | อธิบายจรรยาบรรณวิชาชีพได้ มีความตรงต่อเวลา |
| **PO10** | การบริหารงานวิศวกรรม (Engineering/project management) | [PLO3](CPE_RMUTT68_Prospectus.md#plo3), [PLO8](CPE_RMUTT68_Prospectus.md#plo8) | PLO3: บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาให้กับอุตสาหกรรมหรือชุมชน เพื่อการพัฒนาที่ยั่งยืน · PLO8: ทำงานเป็นทีม วางแผนและบริหารจัดการงานที่ได้รับมอบหมาย สื่อสารความคืบหน้าและปัญหาได้ตรงประเด็น และเสนอแนวทางแก้ไขพร้อมเหตุผลรองรับได้ |
