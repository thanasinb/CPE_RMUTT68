# 04-620-201 ปฏิบัติการควบคุมเวอร์ชัน
## Version Control Laboratory | 1(0-3)

**ประเภท:** กลุ่มวิชาพื้นฐาน | **ชั้นปี:** 2 | **ภาคการศึกษา:** 1

**วิชาบังคับก่อน | Prerequisites:** ไม่มี (None)

---

## คำอธิบายรายวิชา | Course Description

ปฏิบัติการการควบคุมเวอร์ชัน พื้นฐานคำสั่งลินุกซ์ เชลล์สคริปต์ การติดตั้งเครื่องมือควบคุมเวอร์ชัน พื้นที่เก็บข้อมูล การเปลี่ยนแปลง คำสั่งพื้นฐานเช่น คอมมิท พุช พูล พูลรีเควส การเมิร์จ การบรานช์ การทำงานร่วมกัน กรณีศึกษาการควบคุมเวอร์ชันและการใช้ปัญญาประดิษฐ์ช่วยทำงาน / Laboratories on version control; basic Linux command; shell script; tool installation; repositories; changes; basic commands: commit, push, pull; pull request; merging; branching; collaboration; case study in version control and AI-assisted working
