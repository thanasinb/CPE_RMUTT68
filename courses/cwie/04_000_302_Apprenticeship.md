# 04-000-302 ฝึกงาน
## Apprenticeship | 3(0-20)

**ประเภท:** กลุ่มเสริมสร้างประสบการณ์ในวิชาชีพ | **ชั้นปี:** 4 | **ภาคการศึกษา:** 1

**วิชาบังคับก่อน | Prerequisites:** [04-000-301 การเตรียมความพร้อมฝึกประสบการณ์วิชาชีพ (Preparation for Professional Experience)](04_000_301_Professional_Experience_Preparation.md)

---

## คำอธิบายรายวิชา | Course Description

การปฏิบัติงานเต็มเวลาที่ตรงกับสมรรถนะอาชีพในสาขาวิชากำหนดอย่างเป็นระบบ ในสถานประกอบการไม่น้อยกว่า 8 สัปดาห์ และการรายงานผลลัพธ์การเรียนรู้จากการปฏิบัติจริง / Full-time work in the workplace based on specified field of study that is methodically linked to professional competencies for a minimum of 8 weeks with the learning outcomes report reflected from actual practice / หมายเหตุ : การประเมินผลเป็น S และ U

---

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | ประยุกต์ใช้ความรู้ภาคทฤษฎีและปฏิบัติในการฝึกงานทางด้านวิศวกรรมคอมพิวเตอร์ | [PLO3](../../CPE_RMUTT68_Prospectus.md#plo3) |
| CLO2 | เข้าฝึกงานในสถานประกอบการและส่งงานที่ได้รับมอบหมายอย่างสม่ำเสมอและตรงเวลา | [PLO6](../../CPE_RMUTT68_Prospectus.md#plo6) |
| CLO3 | ปรับตัวและเรียนรู้เทคโนโลยีและเครื่องมือใหม่ ๆ ในสถานประกอบการ | [PLO7](../../CPE_RMUTT68_Prospectus.md#plo7) |
| CLO4 | ทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | [PLO8](../../CPE_RMUTT68_Prospectus.md#plo8) |

## ผลลัพธ์การเรียนรู้ระดับหลักสูตร (PLOs) ที่รายวิชารับผิดชอบ

| ผลลัพธ์การเรียนรู้ของหลักสูตร (PLOs) | ความรู้ (K) | ทักษะ (S) | จริยธรรม (E) | ลักษณะบุคคล (C) |
| --- | :---: | :---: | :---: | :---: |
| [PLO3](../../CPE_RMUTT68_Prospectus.md#plo3): บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาให้กับอุตสาหกรรมหรือชุมชน เพื่อการพัฒนาที่ยั่งยืน | ✓ |  |  |  |
| [PLO6](../../CPE_RMUTT68_Prospectus.md#plo6): อธิบายจรรยาบรรณวิชาชีพได้ มีความตรงต่อเวลา |  |  | ✓ |  |

| [PLO8](../../CPE_RMUTT68_Prospectus.md#plo8): ทำงานเป็นทีม วางแผนและบริหารจัดการงานที่ได้รับมอบหมาย สื่อสารความคืบหน้าและปัญหาได้ตรงประเด็น และเสนอแนวทางแก้ไขพร้อมเหตุผลรองรับได้ |  |  |  | ✓ |
