# 04-620-303 การเตรียมโครงงานวิศวกรรมคอมพิวเตอร์
## Computer Engineering Pre-Project | 1(0-3)

**ประเภท:** กลุ่มวิชาบังคับ | **ชั้นปี:** 3 | **ภาคการศึกษา:** 2

**วิชาบังคับก่อน | Prerequisites:** ไม่มี (None)

---

## คำอธิบายรายวิชา | Course Description

วิธีดำเนินงานโครงงานทางวิศวกรรมคอมพิวเตอร์ การเลือกและศึกษาความเป็นไปได้ของข้อเสนอโครงงาน การศึกษาความเป็นมาของปัญหา การกำหนดวัตถุประสงค์และขอบเขต การวางแผนโครงงาน วัสดุและอุปกรณ์ที่ต้องการ การรายงานความคืบหน้า การนำเสนอหัวข้อโครงงาน / Project methodologies of a project in the field of Computer Engineering; selecting and feasible study of project proposal; problem background study; defining objectives and scopes; project planning; required materials and equipment; progress report; proposal presentation
