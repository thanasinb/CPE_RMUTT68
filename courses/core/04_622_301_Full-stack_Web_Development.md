## คำอธิบายรายวิชา | Course Description

การออกแบบและการสร้างเว็บไซต์และแอพพลิเคชั่น การพัฒนาส่วนหน้าและการพัฒนาส่วนหลัง กระบวนการพัฒนาเว็บไซต์และแอพพลิเคชั่น การพัฒนาและการบำรุงรักษาบริการเว็บและอินเทอร์เฟซ / Design and create websites and applications; basic front-end and back-end development; website and application development process; developing and maintaining web services and interfaces
