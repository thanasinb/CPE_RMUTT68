## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | อธิบายและประยุกต์ใช้หลักการ สถาปัตยกรรม การเชื่อมต่อและโปรโตคอลที่เกี่ยวข้องกับอุปกรณ์และระบบอินเตอร์เน็ตของสรรพสิ่งได้ | PLO2 |
| CLO2 | สามารถออกแบบและพัฒนาระบบอินเตอร์เน็ตของสรรพสิ่งเพื่อใช้งานในศาสตร์อื่นได้ | PLO3 |
| CLO3 | สามารถวิเคราะห์ความต้องการข้อจำกัดและสถาปัตยกรรมของระบบอินเตอร์เน็ตของสรรพสิ่งได้ | PLO4 |
| CLO4 | สามารถใช้เครื่องมือทางอิเล็กทรอนิกส์และซอฟต์แวร์พัฒนาระบบอินเตอร์เน็ตของสรรพสิ่งได้ | PLO5 |
| CLO5 | สามารถเรียนรู้เทคโนโลยีใหม่ ๆ สำหรับการพัฒนาระบบอินเตอร์เน็ตของสรรพสิ่งได้ | PLO7 |
| CLO6 | ทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | PLO8 |
