# 04-620-302 ปฏิบัติการบูรณาการระบบคอมพิวเตอร์
## Computer System Integration Laboratory | 2(0-6)

**ประเภท:** กลุ่มวิชาบังคับ | **ชั้นปี:** 3 | **ภาคการศึกษา:** 2

**วิชาบังคับก่อน | Prerequisites:** ไม่มี (None)

---

## คำอธิบายรายวิชา | Course Description

ปฏิบัติการการบูรณาการระบบฮาร์ดแวร์และซอฟต์แวร์ การออกแบบอินเตอร์เฟซระบบปฏิบัติการ อันตรกิริยาฮาร์ดแวร์-ซอฟต์แวร์ การออกแบบวงจรฮาร์ดแวร์ การเขียนโปรแกรมสื่อสารระหว่างอุปกรณ์กับเซิร์ฟเวอร์และฐานข้อมูล การแสดงผลข้อมูล การใช้ปัญญาประดิษฐ์ช่วยในการพัฒนาระบบคอมพิวเตอร์ / Practice in integrating hardware and software systems; designing the interface of an operating system; hardware-software interaction; hardware circuit design; programming communication between devices, servers and database; data visualization; AI-assisted computer system development

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถประยุกต์ใช้หลักการและความรู้พื้นฐานทางวิศวกรรมคอมพิวเตอร์ในการปฏิบัติจริงเพื่อออกแบบและพัฒนาระบบคอมพิวเตอร์ได้ | [PLO2](../../CPE_RMUTT68_Prospectus.md#plo2) |
| CLO2 | สามารถบูรณาการระบบคอมพิวเตอร์กับศาสตร์อื่นรวมถึงตรวจสอบและปรับปรุงการทำงานได้ | [PLO3](../../CPE_RMUTT68_Prospectus.md#plo3) |
| CLO3 | สามารถวิเคราะห์ปัญหาที่เกิดขึ้นจากการทำงานของระบบคอมพิวเตอร์บูรณาการและใช้กระบวนการคิดอย่างเป็นระบบในการแก้ไขปัญหาและหาวิธีปรับปรุงประสิทธิภาพของระบบ | [PLO4](../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO4 | สามารถใช้เครื่องมือและอุปกรณ์ทางวิศวกรรมคอมพิวเตอร์ในการ ตรวจสอบ วิเคราะห์และแก้ปัญหาระบบคอมพิวเตอร์บูรณาการได้อย่างเหมาะสมและมีประสิทธิภาพ | [PLO5](../../CPE_RMUTT68_Prospectus.md#plo5) |
| CLO5 | แสดงออกถึงความซื่อสัตย์ มีจรรยาบรรณทางวิชาการ มีวินัย ตรงต่อเวลา | [PLO6](../../CPE_RMUTT68_Prospectus.md#plo6) |
| CLO6 | สามารถทำงานร่วมกับผู้อื่นได้อย่างมีประสิทธิภาพ สื่อสารอย่างสร้างสรรค์ และมีส่วนร่วมในการแก้ไขปัญหาภายในทีม ทั้งในด้านเทคนิคและการบริหารจัดการ | [PLO8](../../CPE_RMUTT68_Prospectus.md#plo8) |
