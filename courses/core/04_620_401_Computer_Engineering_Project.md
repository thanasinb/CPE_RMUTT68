# 04-620-401 โครงงานวิศวกรรมคอมพิวเตอร์
## Computer Engineering Project | 3(1-6)

**ประเภท:** กลุ่มวิชาบังคับ | **ชั้นปี:** 4 | **ภาคการศึกษา:** 2

**วิชาบังคับก่อน | Prerequisites:** [04-620-303 การเตรียมโครงงานวิศวกรรมคอมพิวเตอร์ (Computer Engineering Pre-Project)](04_620_303_Computer_Engineering_Pre-Project.md)

---

## คำอธิบายรายวิชา | Course Description

การทำโครงงานหรืองานวิจัยที่ได้รับอนุมัติ การรายงานความคืบหน้า การนำเสนอผลการดำเนินงาน การจัดทำรายงานโครงงานวิศวกรรมคอมพิวเตอร์ฉบับสมบูรณ์หรือบทความทางวิชาการหรือวารสารทางวิชาการ / Implementation of improved project; progress report; project examination; presentation of completed project; submitting Computer Engineering project report or academic proceeding or academic journal

---

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถส่งผลงานของโครงงานหรือผลงานวิจัยและปริญญานิพนธ์ฉบับสมบูรณ์หรือบทความทางวิชาการหรือวารสารทางวิชาการ | [PLO3](../../CPE_RMUTT68_Prospectus.md#plo3) |
| CLO2 | สามารถวิเคราะห์และออกแบบระบบทางวิศวกรรมคอมพิวเตอร์ตามโจทย์ที่กำหนดได้ | [PLO4](../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO3 | สามารถใช้เครื่องมือที่เกี่ยวข้องได้อย่างถูกต้องและปลอดภัย | [PLO5](../../CPE_RMUTT68_Prospectus.md#plo5) |
| CLO4 | นำเสนอรายงานความคืบหน้าของงานอย่างสม่ำเสมอ | [PLO6](../../CPE_RMUTT68_Prospectus.md#plo6) |
| CLO5 | สามารถศึกษาเทคโนโลยีใหม่ ๆ เพื่อให้โครงงานประสบความสำเร็จ | [PLO7](../../CPE_RMUTT68_Prospectus.md#plo7) |
| CLO6 | สามารถทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | [PLO8](../../CPE_RMUTT68_Prospectus.md#plo8) |
