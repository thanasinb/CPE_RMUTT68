## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถส่งผลงานของโครงงานหรือผลงานวิจัยและปริญญานิพนธ์ฉบับสมบูรณ์หรือบทความทางวิชาการหรือวารสารทางวิชาการ | PLO3 |

| CLO3 | สามารถใช้เครื่องมือที่เกี่ยวข้องได้อย่างถูกต้องและปลอดภัย | PLO5 |
| CLO4 | นำเสนอรายงานความคืบหน้าของงานอย่างสม่ำเสมอ | PLO6 |
| CLO5 | สามารถศึกษาเทคโนโลยีใหม่ ๆ เพื่อให้โครงงานประสบความสำเร็จ | PLO7 |
| CLO6 | สามารถทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | PLO8 |
