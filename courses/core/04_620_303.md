## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถวิเคราะห์และออกแบบระบบทางวิศวกรรมคอมพิวเตอร์ตามโจทย์ที่ตนสนใจได้ | PLO2 |
| CLO2 | สามารถอธิบายปัญหาในศาสตร์อื่นเพื่อนำมาแก้ปัญหาโดยใช้ระบบทางวิศวกรรมคอมพิวเตอร์ได้ | PLO3 |
| CLO3 | สามารถวิเคราะห์ความต้องการของระบบทางวิศวกรรมคอมพิวเตอร์ และวางแผนการดำเนินงานโครงการวิศวกรรมคอมพิวเตอร์ตามโจทย์ที่ตนสนใจได้ | PLO4 |
| CLO4 | นำเสนอรายงานความคืบหน้าและหลักการแก้ไขปัญหาของงานกับอาจารย์ที่ปรึกษาโครงงานอย่างสม่ำเสมอ | PLO6 |
| CLO5 | สามารถเรียนรู้เทคโนโลยีใหม่ ๆ ทางด้านวิศวกรรมคอมพิวเตอร์ได้ | PLO7 |
| CLO6 | สามารถทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | PLO8 |
