# 04-623-301 ปฏิบัติการเครือข่ายคอมพิวเตอร์
## Computer Network Laboratory | 1(0-3)

**ประเภท:** กลุ่มวิชาบังคับ | **ชั้นปี:** 3 | **ภาคการศึกษา:** 1

**วิชาบังคับก่อน | Prerequisites:** ไม่มี (None)

---

## คำอธิบายรายวิชา | Course Description

ปฏิบัติการในเนื้อหาที่เกี่ยวกับเครือข่ายคอมพิวเตอร์ การสวิตช์ เครือข่ายเสมือน เครือข่ายสำรอง การตั้งค่าการกำหนดเส้นทาง / Laboratories on computer network field; switching; VLAN; redundant networks; routing configuration

---

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถตั้งค่าระบบเครือข่ายคอมพิวเตอร์ได้ | [PLO2](../../CPE_RMUTT68_Prospectus.md#plo2) |
| CLO2 | สามารถวิเคราะห์และประเมินปัญหาที่เกิดในเครือข่ายคอมพิวเตอร์ได้ | [PLO4](../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO3 | แสดงออกถึงความซื่อสัตย์ มีจรรยาบรรณทางวิชาการ มีวินัย ตรงต่อเวลา | [PLO6](../../CPE_RMUTT68_Prospectus.md#plo6) |
