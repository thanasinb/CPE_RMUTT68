## คำอธิบายรายวิชา | Course Description

สถาปัตยกรรมเครือข่ายเซ็นเซอร์ไร้สาย สถาปัตยกรรมเซ็นเซอร์ไร้สาย การออกแบบเครือข่ายเซ็นเซอร์ไร้สาย การประยุกต์ใช้งานเครือข่ายเซ็นเซอร์ไร้สาย โพรโทคอลควบคุมการเข้าถึงในเครือข่ายเซ็นเซอร์ไร้สาย เช่น IEEE 802.15.4 6LoWPAN โพรโทคอลหาเส้นทางในเครือข่ายเซ็นเซอร์ไร้สาย เช่น DSR AODV ระบบปฏิบัติการเครือข่ายเซ็นเซอร์ไร้สาย เช่น TinyOS Contiki / Wireless sensor network architecture; sensor architecture; wireless sensor network design; wireless sensor network applications; medium access control protocols in wireless sensor network such as IEEE 802.15.4, 6LoWPAN; routing protocols in wireless sensor network such as DSR, AODV; operating systems in wireless sensor network such as TinyOS Contiki
