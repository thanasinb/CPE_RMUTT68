## คำอธิบายรายวิชา | Course Description

ระบบปฏิบัติการเสมือนยูนิกซ์และการเขียนโปรแกรมในระดับของระบบปฏิบัติการ ประกอบด้วยการเขียนโปรแกรมเชลล์ การใช้งานซิสเต็มคอลล์ ระบบไฟล์และไดเร็กทอรีประมวลผลไฟล์ การสร้างโพรเซส การควบคุมโพรเซส การส่งสัญญาณ การติดต่อกับเทอร์มินอล การสร้างเดมอนโพรเซส และการติดต่อระหว่างโปรแกรม / UNIX-Like operating system with an emphasis on programming at the system level including shells and shell script programming; system calls; file and directory systems; file processing; process; process control; terminal I/O; daemon process; interprocess communication
