## คำอธิบายรายวิชา | Course Description

ทฤษฎีเกม การออกแบบและการพัฒนาเกมด้วยการใช้เครื่องมือพัฒนาเกมเช่น ยูนิตี้ หรือ อันเรียลเอ็นจิน การสร้างเกมในระบบความจริงเสมือน การใช้เอพีไอหรือไลบรารีเพื่อควบคุมและตรวจจับการกระทำของตัวละคร การตรวจจับการชนของวัตถุ การจัดการกับเหตุการณ์และการเคลื่อนไหวในเกม กระบวนการทดสอบและปรับปรุงเกม / Game theory; design and development with game development engine for instance Unity, Unreal Engine; game development in virtual reality (VR); using API or libraries for controlling and detecting players’ actions; object collision detection; game event control management; test and improvement the developed games
