## คำอธิบายรายวิชา | Course Description

การทดสอบซอฟต์แวร์ ระดับการทดสอบ ชนิดการทดสอบ เทคนิคการทดสอบ การตรวจสอบ การสร้างการทดสอบ เครื่องมือในการทดสอบ การวางแผนและการจัดการการทดสอบ วิธีเชิงรูปนัย การวิเคราะห์คุณภาพซอฟต์แวร์ / Software testing; test levels; test types; testing techniques; inspection; test implementation; test tools; test planning and management; formal methods; software quality analysis
