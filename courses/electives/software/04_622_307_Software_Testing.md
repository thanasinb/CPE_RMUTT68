# 04-622-307 การทดสอบซอฟต์แวร์
## Software Testing | 3(2-3)

**ประเภท:** กลุ่มวิชาเลือก | **ชั้นปี:** N/A (elective) | **ภาคการศึกษา:** N/A

**วิชาบังคับก่อน | Prerequisites:** ไม่มี (None)

---

## คำอธิบายรายวิชา | Course Description

การทดสอบซอฟต์แวร์ ระดับการทดสอบ ชนิดการทดสอบ เทคนิคการทดสอบ การตรวจสอบ การสร้างการทดสอบ เครื่องมือในการทดสอบ การวางแผนและการจัดการการทดสอบ วิธีเชิงรูปนัย การวิเคราะห์คุณภาพซอฟต์แวร์ / Software testing; test levels; test types; testing techniques; inspection; test implementation; test tools; test planning and management; formal methods; software quality analysis

---

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถอธิบายรูปแบบการทดสอบซอฟต์แวร์ในตามวัตถุประสงค์การออกแบบซอฟต์แวร์ | [PLO1](../../../CPE_RMUTT68_Prospectus.md#plo1) |
| CLO2 | สามารถอธิบายกายภาพของหลักการทดสอบซอฟต์แวร์ | [PLO2](../../../CPE_RMUTT68_Prospectus.md#plo2) |
| CLO3 | สามารถประยุกต์ใช้หลักการทดสอบซอฟต์แวร์สำหรับเป็นแนวทางในการออกแบบระบบซอฟต์แวร์ขั้นสูงได้อย่างเหมาะสม | [PLO2](../../../CPE_RMUTT68_Prospectus.md#plo2) |
| CLO4 | สามารถวิเคราะห์ปัญหา ดำเนินการปรับปรุงประสิทธิภาพระบบซอฟต์แวร์ที่ได้ทำการทดสอบ และนำเสนอ | [PLO4](../../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO5 | สามารถยืนยันผลการทดสอบซอฟต์แวร์ ที่เป็นไปตามหลักวิชาการได้ถูกต้อง | [PLO6](../../../CPE_RMUTT68_Prospectus.md#plo6) |
