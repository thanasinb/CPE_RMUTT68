# 04-622-304 การโปรแกรมเกม
## Game Programming | 3(2-3)

**ประเภท:** กลุ่มวิชาเลือก | **ชั้นปี:** N/A (elective) | **ภาคการศึกษา:** N/A

**วิชาบังคับก่อน | Prerequisites:** [04-622-203 การเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming)](../../core/04_622_203_Object-Oriented_Programming.md)

---

## คำอธิบายรายวิชา | Course Description

ทฤษฎีเกม การออกแบบและการพัฒนาเกมด้วยการใช้เครื่องมือพัฒนาเกมเช่น ยูนิตี้ หรือ อันเรียลเอ็นจิน การสร้างเกมในระบบความจริงเสมือน การใช้เอพีไอหรือไลบรารีเพื่อควบคุมและตรวจจับการกระทำของตัวละคร การตรวจจับการชนของวัตถุ การจัดการกับเหตุการณ์และการเคลื่อนไหวในเกม กระบวนการทดสอบและปรับปรุงเกม / Game theory; design and development with game development engine for instance Unity, Unreal Engine; game development in virtual reality (VR); using API or libraries for controlling and detecting players’ actions; object collision detection; game event control management; test and improvement the developed games

| CLO2 | สามารถพัฒนาส่วนของโปรแกรมที่ได้รับมอบหมายและนำมาประกอบรวมกันเพื่อให้ได้งานที่สมบูรณ์ได้ | [PLO2](../../../CPE_RMUTT68_Prospectus.md#plo2) |

| CLO4 | สามารถวิเคราะห์ ร่างภาพ จำแนกความแตกต่างของของตัวละคร ไอเท็ม สภาพแวดล้อม สถานการณ์และทรัพยากรได้ | [PLO4](../../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO5 | ผู้เรียนสามารถใช้เครื่องมือในการพัฒนาโปรแกรมตามที่ได้รับมอบหมายได้ | [PLO5](../../../CPE_RMUTT68_Prospectus.md#plo5) |
| CLO6 | แสดงออกถึงความซื่อสัตย์ มีจรรยาบรรณทางวิชาการ มีวินัย ตรงต่อเวลา | [PLO6](../../../CPE_RMUTT68_Prospectus.md#plo6) |
