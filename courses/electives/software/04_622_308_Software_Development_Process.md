# 04-622-308 กระบวนการพัฒนาซอฟต์แวร์
## Software Development Process | 3(2-3)

**ประเภท:** กลุ่มวิชาเลือก | **ชั้นปี:** N/A (elective) | **ภาคการศึกษา:** N/A

**วิชาบังคับก่อน | Prerequisites:** [04-622-301 การพัฒนาเว็บเต็มรูปแบบ (Full-stack Web Development)](../../core/04_622_301_Full-stack_Web_Development.md)

## คำอธิบายรายวิชา | Course Description

การจัดการประสิทธิภาพของแอปพลิเคชันและการบูรณาการอย่างต่อเนื่อง กลยุทธ์การใช้งานอย่างต่อเนื่อง การเขียนโปรแกรม และการพัฒนาเว็บ สแต็กเทคโนโลยี / Application performance management and continuous integration; continuous deployment strategies; programming and web development; technology stack

---

## ผลลัพธ์การเรียนรู้รายวิชา | Course Learning Outcomes (CLOs)

| CLO | ผลลัพธ์การเรียนรู้ | PLOs |
| --- | --- | --- |
| CLO1 | สามารถประยุกต์ใช้หลักการและเทคนิคในการจัดการประสิทธิภาพของแอปพลิเคชันและการบูรณาการอย่างต่อเนื่องโดยใช้สแต็กเทคโนโลยี | [PLO2](../../../CPE_RMUTT68_Prospectus.md#plo2) |
| CLO2 | สามารถบูรณาการความรู้ในการพัฒนาแอปพลิเคชันและเว็บเพื่อแก้ไขปัญหาทางวิศวกรรมและสนับสนุนการพัฒนาที่ยั่งยืน | [PLO3](../../../CPE_RMUTT68_Prospectus.md#plo3) |
| CLO3 | วิเคราะห์และพัฒนากลยุทธ์การใช้งานอย่างต่อเนื่องและการเขียนโปรแกรมที่มีประสิทธิภาพ | [PLO4](../../../CPE_RMUTT68_Prospectus.md#plo4) |
| CLO4 | สามารถวิเคราะห์และประเมินความเหมาะสมของเครื่องมือ ทรัพยากร เทคโนโลยีที่ต้องใช้กับโครงการ | [PLO4](../../../CPE_RMUTT68_Prospectus.md#plo4) |
