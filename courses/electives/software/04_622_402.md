# 04-622-402 ฐานข้อมูลไม่สัมพันธ์
## Non-Relational Database | 3(2-3)

**ประเภท:** กลุ่มวิชาเลือก | **ชั้นปี:** N/A (elective) | **ภาคการศึกษา:** N/A

---

## คำอธิบายรายวิชา | Course Description

แนวคิดหลักและลักษณะของฐานข้อมูลไม่สัมพันธ์ ความแตกต่างจากฐานข้อมูลเชิงสัมพันธ์ ชนิดของการจัดเก็บ การรวมตัวแบบข้อมูล ฐานข้อมูลแบบ key-value ฐานข้อมูลแบบ document ฐานข้อมูลแบบ column-oriented ฐานข้อมูลแบบ graph การแปลงโครงสร้าง การประยุกต์ใช้งาน NoSQL / NoSQL concepts and characteristics of non-relational databases; differences from relational databases; NoSQL storage types; aggregate data models; implementation of NoSQL; key- value databases; document database; column-oriented database; graph databases; schema migration; NoSQL database application
