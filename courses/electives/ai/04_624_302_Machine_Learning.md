## คำอธิบายรายวิชา | Course Description

การเรียนรู้ของเครื่อง การเรียนรู้แบบมีผู้สอนและไม่มีผู้สอน การจำแนกประเภท การจัดกลุ่ม ต้นไม้ตัดสินใจ เพื่อนบ้านใกล้ที่สุด โครงข่ายประสาทเทียม เบย์ ซัพพอร์ทเวกเตอร์แมชชีน การลดมิติข้อมูล การตรวจสอบไขว้ การประเมินผลของการทำนาย / Machine learning; supervised and unsupervised learning; classification; clustering; decision tree; K-nearest neighbors; artificial neural networks; Bayesian Learning; support vector machine; dimensionality reduction; cross-validation; evaluating estimator performance
