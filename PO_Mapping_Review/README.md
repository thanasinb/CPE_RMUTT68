# PLO4 / PLO8 → TABEE PO Mapping Review

หลักสูตรวิศวกรรมคอมพิวเตอร์ (ปรับปรุง พ.ศ. 2568) เดิมพัฒนาโดยอ้างอิงมาตรฐาน AUN-QA ซึ่งกำหนดผลลัพธ์การเรียนรู้ระดับหลักสูตร (PLO) เท่านั้น ยังไม่มีการระบุ TABEE Program Outcome (PO) ในการเตรียมยื่นขอรับรอง TABEE (Sydney Accord) จำเป็นต้องระบุ PO ที่ชัดเจนสำหรับทุก CLO ของทุกรายวิชา

ตาราง PLO Mapping ของทุกรายวิชาเดิมมีปัญหาว่า **PLO4** (→ PO2, PO4) และ **PLO8** (→ PO6, PO7, PO10) แต่ละ PLO ครอบคลุมมากกว่า 1 TABEE PO ทำให้อาจารย์ผู้สอนไม่ทราบแน่ชัดว่า CLO ของตนต้องรับผิดชอบ PO ข้อใดบ้าง เอกสารชุดนี้จึงแยกคอลัมน์ PLO4/PLO8 ออกเป็นคอลัมน์ย่อยตาม PO พร้อมเหตุผลของการจัดกลุ่ม โดยวิเคราะห์จากข้อความ CLO จริงของแต่ละรายวิชาเทียบกับนิยาม PO อย่างเป็นระบบ **นี่คือข้อเสนอเบื้องต้น ไม่ใช่ข้อสรุปสุดท้าย** — แต่ละรายวิชามีไฟล์แยกสำหรับส่งให้อาจารย์ผู้สอนตรวจทานและแจ้งกลับ

## หลักเกณฑ์การจำแนกโดยสรุป

- **PLO4 → PO2 vs PO4**: ค่าเริ่มต้นคือ PO2 (การวิเคราะห์ปัญหา) เนื่องจาก CLO ส่วนใหญ่ใช้ถ้อยคำ “วิเคราะห์...ปัญหา/ระบบ” จะเพิ่ม PO4 (การพิจารณาตรวจสอบ) เฉพาะเมื่อ CLO ระบุการทดสอบ/ทดลอง/จำลองสถานการณ์/ประเมินผลแบบจำลองอย่างชัดเจนเท่านั้น (มี 5 รายวิชาที่เข้าเกณฑ์นี้)
- **PLO8 → PO6 / PO7 / PO10**: CLO ส่วนใหญ่ใช้ประโยคสำเร็จรูป “ทำงานเป็นทีม สามารถสื่อสาร...” ซึ่งครอบคลุมทั้ง PO6 และ PO7 ในประโยคเดียว จึงทำเครื่องหมายทั้งสอง PO ตามค่าเริ่มต้น ยกเว้น 2 รายวิชาที่มีถ้อยคำต่างออกไป (ดูตารางด้านล่าง)

## ข้อสังเกตเชิงหลักสูตรที่ควรทราบก่อนตรวจทาน

- **PO4 (Investigation)** มี CLO รองรับเพียง 5 รายวิชาทั้งหลักสูตร ส่วนใหญ่ของ PLO4 ตกอยู่ที่ PO2 เพียงอย่างเดียว — หากคณะกรรมการ TABEE ต้องการเห็น PO4 ที่ครอบคลุมกว่านี้ อาจต้องพิจารณาปรับ/เพิ่ม CLO ในบางรายวิชาปฏิบัติการหรือรายวิชาที่มีการทดลอง/สร้างต้นแบบ
- **PO10 (Project/Engineering Management)** มี CLO ระบุชัดเจนเพียง **1 รายวิชา** ทั้งหลักสูตร (04-620-302 ปฏิบัติการบูรณาการระบบคอมพิวเตอร์) แม้แต่รายวิชาโครงงาน (04-620-303, 04-620-401) ก็ยังไม่มีถ้อยคำด้านการบริหารจัดการโครงการอย่างชัดเจนใน CLO — เป็นช่องว่างที่ควรพิจารณาแก้ไข
- CLO ที่ครอบคลุม 2 PO ในประโยคเดียว (ส่วนใหญ่ในกลุ่ม PLO8) ยังไม่ได้ถูกแยกเป็น CLO ย่อย — เป็นทางแก้ระยะสั้นที่ไม่กระทบการนับ/เรียงลำดับ CLO เดิม หากต้องการความชัดเจนต่อ PO แบบ 1 CLO : 1 PO ในอนาคต จะต้องแก้ไขข้อความ CLO และปรับเอกสาร OBE3-TABEE เป็นรายวิชา
