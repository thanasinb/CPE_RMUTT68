## ผลลัพธ์การเรียนรู้ระดับหลักสูตร (เวอร์ชันเดิม)

PLO เหล่านี้ถูกใช้ใน `CPE_RMUTT68_Prospectus.md` ก่อนการปรับปรุงครั้งล่าสุด (มีผลตั้งแต่ 1 กรกฎาคม 2568)

| PLO | ผลลัพธ์การเรียนรู้ (เดิม) | ด้าน |
| --- | --- | --- |
| PLO1 | อธิบายและคำนวณหลักการสำคัญทางคณิตศาสตร์พื้นฐาน วิทยาศาสตร์พื้นฐาน วิศวกรรมศาสตร์พื้นฐาน และวิศวกรรมคอมพิวเตอร์พื้นฐานได้ | K |
| PLO2 | ประยุกต์ใช้ความรู้ทั้งในเชิงทฤษฎีและปฏิบัติในเนื้อหาของสาขาวิชาเฉพาะด้านทางวิศวกรรมคอมพิวเตอร์ได้ | K |
| PLO3 | บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาในศาสตร์อื่น ๆ เพื่อการพัฒนาที่ยั่งยืน | K |
| PLO4 | คิดวิเคราะห์อย่างเป็นระบบ และแก้ปัญหาทางวิศวกรรมคอมพิวเตอร์และเทคโนโลยีได้ | S |
| PLO5 | ใช้เครื่องมือทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างถูกต้องเหมาะสม | S |
| PLO6 | แสดงออกถึงความซื่อสัตย์ มีจรรยาบรรณทางวิชาการ มีวินัย ตรงต่อเวลา | E |
| PLO7 | สามารถเรียนรู้เทคโนโลยีใหม่ ๆ ทางด้านวิศวกรรมคอมพิวเตอร์ได้ | C |
| PLO8 | ทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | C |

## การเปลี่ยนแปลงจาก PLO เดิม → ใหม่

| PLO | เดิม | ใหม่ |
| --- | --- | --- |
| PLO1 | อธิบายและคำนวณหลักการสำคัญทางคณิตศาสตร์พื้นฐาน วิทยาศาสตร์พื้นฐาน วิศวกรรมศาสตร์พื้นฐาน และวิศวกรรมคอมพิวเตอร์พื้นฐานได้ | *(เหมือนเดิม)* |
| PLO2 | ประยุกต์ใช้ความรู้ทั้งในเชิงทฤษฎีและปฏิบัติในเนื้อหาของสาขาวิชาเฉพาะด้านทางวิศวกรรมคอมพิวเตอร์ได้ | ใช้ความรู้และทักษะ ในเชิงทฤษฎีและปฏิบัติ เพื่อแก้ไขปัญหาด้านวิศวกรรมคอมพิวเตอร์ได้ |
| PLO3 | บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาในศาสตร์อื่น ๆ เพื่อการพัฒนาที่ยั่งยืน | บูรณาการความรู้ในสาขาวิศวกรรมคอมพิวเตอร์เพื่อแก้ไขปัญหาให้กับอุตสาหกรรมหรือชุมชน เพื่อการพัฒนาที่ยั่งยืน |
| PLO4 | คิดวิเคราะห์อย่างเป็นระบบ และแก้ปัญหาทางวิศวกรรมคอมพิวเตอร์และเทคโนโลยีได้ | คิดวิเคราะห์อย่างเป็นระบบ พัฒนาระบบงานทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างเป็นขั้นตอน |
| PLO5 | ใช้เครื่องมือทางด้านวิศวกรรมคอมพิวเตอร์ได้อย่างถูกต้องเหมาะสม | *(เหมือนเดิม)* |
| PLO6 | แสดงออกถึงความซื่อสัตย์ มีจรรยาบรรณทางวิชาการ มีวินัย ตรงต่อเวลา | อธิบายจรรยาบรรณวิชาชีพได้ มีความตรงต่อเวลา |
| PLO7 | สามารถเรียนรู้เทคโนโลยีใหม่ ๆ ทางด้านวิศวกรรมคอมพิวเตอร์ได้ | อธิบายและประยุกต์ใช้เทคโนโลยีทางวิศวกรรมคอมพิวเตอร์ที่ไม่เคยเรียนในหลักสูตรได้ |
| PLO8 | ทำงานเป็นทีม สามารถสื่อสารและแสดงความคิดเห็นได้อย่างสร้างสรรค์ | ทำงานเป็นทีม สื่อสารความคืบหน้าและปัญหาได้ตรงประเด็น และเสนอแนวทางแก้ไขพร้อมเหตุผลรองรับได้ |
